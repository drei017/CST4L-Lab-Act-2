# CST4L-Lab-Act-2
NFA Simulation (Multiple Next States) Objective: Build an NFA simulator for strings containing ab as a substring.  Instructions:  Represent the NFA using a map: state -> (symbol -> set of next states). Accept strings that contain the substring "ab". Use recursion or queue to explore multiple transitions.

Example Input/Output
Input: aab
Output: Accepted

Input: bbaa
Output: Rejected
